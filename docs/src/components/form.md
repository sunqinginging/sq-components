## Form API

### Form Attributes

| 属性名 |                                        说明                                        |  类型  | 默认值 |
| :----: | :--------------------------------------------------------------------------------: | :----: | :----: |
| model  |                                    表单数据对象                                    | object |   --   |
| rules  | 表单校验规则,使用[async-validator 库](https://github.com/yiminghe/async-validator) | object |   --   |

### Form Exposes

|    名称     |                                                                说明                                                                |
| :---------: | :--------------------------------------------------------------------------------------------------------------------------------: |
|  validate   |                                             对整个表单的内容进行校验,返回一个 promise                                              |
| resetFields | 重置该表单项，将其值重置为初始值，并移除校验结果。若传数组参数，则对数组内的字段进行重置和移除校验，不传或者传空数组则对所有的字段 |
