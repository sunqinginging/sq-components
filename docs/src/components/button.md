## API

### Button 属性

|  属性名  |      说明      |                  类型                   | 默认值  |
| :------: | :------------: | :-------------------------------------: | :-----: |
|   size   |      尺寸      |          defalut、large、small          | default |
|   type   |    按钮类型    | primary、success、warning、info、danger | primary |
| disabled |  按钮是否禁用  |                 boolean                 |  false  |
|   text   | 是否为文字按钮 |                 boolean                 |  false  |
|  plain   | 是否为朴素按钮 |                 boolean                 |  false  |
